== Description ==

This plugin sends out your mails asynchronous, so you and your visitors have not to wait for a response of a slow SMTP server

== Installation ==

1. Upload `cubetech-async-mailer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
